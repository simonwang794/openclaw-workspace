## 🚀 下一步：執行部署

### 方式一：一鍵部署（推薦）

```bash
cd ~/.openclaw/workspace
./deploy_automation.sh
```

### 方式二：手動部署

1. **更新 .env**
   ```bash
   cd ~/.gemini/antigravity/scratch/financial_assistant
   nano .env
   # 新增 TELEGRAM_BOT_TOKEN 和 TELEGRAM_CHAT_ID
   ```

2. **建立目錄**
   ```bash
   mkdir -p logs/archive
   mkdir -p scripts
   ```

3. **安裝 launchd**
   ```bash
   cp ~/.openclaw/workspace/*.plist ~/Library/LaunchAgents/
   launchctl load ~/Library/LaunchAgents/com.simon.financial-assistant.plist
   launchctl load ~/Library/LaunchAgents/com.simon.financial-assistant.reminders.plist
   ```

4. **測試**
   ```bash
   python3 modules/telegram_notifier.py
   python3 main_enhanced.py scan
   ```
