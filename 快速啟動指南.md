## 🚀 三步驟快速啟動

### 第一步：環境設定（5 分鐘）

```bash
# 1. 進入專案目錄
cd ~/.gemini/antigravity/scratch/financial_assistant

# 2. 編輯 .env 檔案，新增以下設定
nano .env

# 新增內容：
TELEGRAM_BOT_TOKEN=你的_BOT_TOKEN
TELEGRAM_CHAT_ID=你的_CHAT_ID
NOTION_DASHBOARD_PAGE_ID=你的儀表板頁面_ID（選填）

# 3. 儲存並退出（Ctrl+O, Enter, Ctrl+X）
```

**如何取得 Telegram Bot Token 和 Chat ID？**

1. **建立 Bot：**
   - 在 Telegram 搜尋 `@BotFather`
   - 發送 `/newbot` 指令
   - 按照指示建立 Bot，取得 Token

2. **取得 Chat ID：**
   - 在 Telegram 搜尋 `@userinfobot`
   - 發送任意訊息，Bot 會回覆你的 Chat ID

## 🔧 常用指令

### 管理 launchd 服務

```bash
# 檢查服務狀態
launchctl list | grep financial-assistant

# 手動觸發主掃描
launchctl start com.simon.financial-assistant

# 手動觸發提醒
launchctl start com.simon.financial-assistant.reminders

# 停止服務
launchctl unload ~/Library/LaunchAgents/com.simon.financial-assistant.plist

# 重新載入服務
launchctl unload ~/Library/LaunchAgents/com.simon.financial-assistant.plist
launchctl load ~/Library/LaunchAgents/com.simon.financial-assistant.plist
```

### 查看日誌

```bash
# 即時監看主掃描日誌
tail -f ~/.gemini/antigravity/scratch/financial_assistant/logs/stdout.log

# 查看錯誤日誌
tail -f ~/.gemini/antigravity/scratch/financial_assistant/logs/stderr.log

# 查看提醒日誌
tail -f ~/.gemini/antigravity/scratch/financial_assistant/logs/reminders_stdout.log

# 檢查日誌錯誤
bash ~/.gemini/antigravity/scratch/financial_assistant/scripts/check_logs.sh
```

### 手動執行


## 🛠️ 故障排除

### 問題 1：launchd 服務無法啟動

```bash
# 檢查 plist 語法
plutil -lint ~/Library/LaunchAgents/com.simon.financial-assistant.plist

# 檢查 Python 路徑
which python3

# 確認路徑是否與 plist 中一致
```

### 問題 2：Telegram 通知無法發送

```bash
# 測試 Bot Token
curl "https://api.telegram.org/bot你的_TOKEN/getMe"

# 測試發送訊息
curl -X POST "https://api.telegram.org/bot你的_TOKEN/sendMessage" \
  -d "chat_id=你的_CHAT_ID" \
  -d "text=測試"
```

### 問題 3：Notion API 錯誤

```bash
# 檢查 Token
curl -X GET "https://api.notion.com/v1/users/me" \
  -H "Authorization: Bearer 你的_TOKEN" \
  -H "Notion-Version: 2022-06-28"
```

### 問題 4：日誌顯示權限錯誤

```bash
# 重新設定權限
chmod +x ~/.gemini/antigravity/scratch/financial_assistant/scripts/*.sh
chmod +x ~/.gemini/antigravity/scratch/financial_assistant/scripts/*.py
```
